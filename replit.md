# MediVerse

A healthcare web application built with React + Vite.

## Project Structure

- `frontend/` — React + Vite frontend application
  - `src/pages/` — Page components (Home, Appointments, Doctors, MedicalRecords, Pharmacy)
  - `src/components/` — Shared components (Layout, Navbar)

## Running the App

The frontend runs on port 5000:
```
cd frontend && npm run dev
```
